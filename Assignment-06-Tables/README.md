# Assignment 06 - Tables

## 📚 About This Assignment
Learn to create and structure HTML tables for organizing and displaying tabular data effectively.

## 🔗 Repository Link
**[View Full Assignment Repository](https://github.com/ansarwaseem/Saylani-MWD-Batch-23-Ansar-Waseem-Assignment-06-Tables)**

## 📝 Topics Covered
- Table elements (`<table>`, `<tr>`, `<td>`, `<th>`)
- Table structure (thead, tbody, tfoot)
- Table attributes (colspan, rowspan)
- Styling tables with HTML
- Accessible table practices

## 🎯 Learning Objectives
- Understand HTML table structure and semantics
- Create well-organized tables for data presentation
- Use table headers and proper markup
- Apply colspan and rowspan attributes
- Follow accessibility best practices for tables
